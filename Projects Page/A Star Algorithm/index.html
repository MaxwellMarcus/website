The A* Pathfinding Algorithm is a method of finding a path from a start node to an end node.  A node is an available space that can be used in a path. In the above example everything except walls (the light gray blocks) are nodes. A* starts with a list of nodes that has only the start node in it. Then the algorithm adds the adjacent nodes (not including diagonals in the above example) to the list and assigns each of them a score. The score is the length of the path that the nodes are on plus the euclidean distance from the end node. Finally the algorithm sorts the list of nodes based on each node’s score (lowest score to highest score). The algorithm then repeats this process, but it expands on the node with the lowest score (also the first node in the list) instead of the start node. The algorithm ends when the end node is the node with the lowest score.

In the above example the expanded nodes are all the nodes that have already added their adjacent nodes to the list.

The expanding nodes are all the nodes that are in the list, but have not yet added their adjacent nodes to the list.
